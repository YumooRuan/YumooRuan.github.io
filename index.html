GitHub Pages博客编辑指南

2023年10月25日 技术指南 阅读时间 5分钟

欢迎来到我的GitHub博客！这个博客完全托管在GitHub Pages上，你可以直接在GitHub网页界面上编辑内容。

快速编辑指南

按照以下步骤编辑你的博客内容：

登录GitHub

访问 github.com 并登录你的账号

打开博客仓库

找到你的博客仓库（通常名为 你的用户名.github.io）

编辑index.html文件

点击index.html文件 > 点击右上角的铅笔图标

修改内容

在编辑器中修改HTML内容（下面有详细指南）

提交更改

滚动到底部 > 填写提交信息 > 点击"Commit changes"

添加新文章

要添加新文章，只需复制以下HTML结构并修改内容：

<article class="post">
  <h1 class="post-title">文章标题</h1>
  <div class="post-meta">
    <span><i class="far fa-calendar"></i> 日期</span>
    <span><i class="far fa-folder"></i> 分类</span>
    <span><i class="far fa-clock"></i> 阅读时间</span>
  </div>
  
  <div class="post-content">
    <p>这里是文章内容...</p>
    
    <h2>二级标题</h2>
    <p>更多内容...</p>
    
    <img src="图片URL" alt="图片描述">
  </div>
</article>

编辑技巧

内容编辑提示

添加段落

<p>这是一个新段落</p>

添加图片

<img src="图片URL" alt="图片描述">

添加链接

<a href="https://example.com">链接文本</a>

添加列表

<ul>
  <li>项目1</li>
  <li>项目2</li>
</ul>

添加代码块

<pre><code>你的代码在这里</code></pre>

免费图片资源

为你的博客添加高质量图片：

  • Unsplash: https://unsplash.com
  • Pexels: https://pexels.com
  • Pixabay: https://pixabay.com
